NOTA TÉCNICA:
════════════════════════════════════════════════════════════════════════════

Ahora el script:
1. Detecta la interfaz por defecto con: ip route | grep '^default'
2. Obtiene la IP con: ip addr show <iface>
3. Configura UFW para multicast: ufw allow to 224.0.0.0/4
4. Todo sin intervención del usuario

Esto es tan TRANSPARENTE como Windows, que permite multicast por defecto.
